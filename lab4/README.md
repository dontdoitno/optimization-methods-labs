# Лабораторная работа 4: Управление инвестиционным портфелем методом стохастического динамического программирования

## Описание

Решение задачи управления портфелем ценных бумаг и депозитов в условиях неопределенности методом стохастического динамического программирования с использованием критерия Байеса (максимизация математического ожидания дохода).

## Структура проекта

```
lab4/
├── src/
│   ├── __init__.py          # Инициализация пакета
│   ├── models.py            # Модели данных (PortfolioState, Action, Scenario, Stage)
│   └── solver.py            # Реализация алгоритма ДП (InvestmentSolver)
├── main.py                  # Главный файл для запуска программы
├── report.md                # Подробный отчет с математической моделью
└── README.md                # Этот файл
```

## Требования

- Python 3.8+

## Запуск программы

```bash
cd lab4
python main.py
```

## Входные данные

**Начальное состояние портфеля:**
- ЦБ1: 100 д.е.
- ЦБ2: 800 д.е.
- Депозиты: 400 д.е.
- Свободные средства: 600 д.е.

**Пакеты для операций:**
- Пакет ЦБ1 = 25 д.е. (1/4 от начальной стоимости)
- Пакет ЦБ2 = 200 д.е. (1/4 от начальной стоимости)
- Пакет Деп = 100 д.е. (1/4 от начальной стоимости)

**Период планирования:** 3 этапа с различными вероятностями и коэффициентами изменения стоимости активов.

## Результаты

Программа выводит:
1. Начальное состояние портфеля
2. Описание этапов планирования
3. Оптимальную стратегию управления по этапам
4. Максимальный ожидаемый доход
5. Пример оптимального пути для одного из сценариев
6. Статистику решения

## Метод решения

- **Метод:** Стохастическое динамическое программирование (обратный ход)
- **Критерий:** Критерий Байеса (максимизация математического ожидания дохода)
- **Алгоритм:** Уравнения Беллмана с кэшированием состояний (мемоизация)
